# 성공 시나리오

### 사전 조건 :

 

1. 사용자가 플레이어의 햄버거 버튼을 누르고 타이머 아이콘을 클릭한다.
2. 시간으로 타이머를 설정할 수 있는 모달이 표시된다. (최대 2시간 + 지금 재생 중인 노래 종료) (분)

### 사후 조건
