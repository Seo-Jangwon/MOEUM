# 성공 시나리오

### 사전 조건 :

- 로그인

1. **왼쪽 사이드바**에서 ‘내가 본 음악’ 메뉴를 선택한다.
2. ‘내가 본 음악’ 페이지로 이동한다.

### 사후 조건:

- 아직 재생 기록이 없는 경우, “아직 재생 기록이 없습니다.” 문구를 표시한다.

- FE
    - 내 재생 기록 조회 API를 호출한다.
- BE
    - API 결과를 반환한다.
